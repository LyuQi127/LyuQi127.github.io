**Awesome World Models** is a curated repository accompanying the survey paper _Learning to Model the World: A Survey of World Models in Artificial Intelligence_. I'm honored to be a contributor of this project about world models.

It organizes world model research across multiple directions, including reinforcement learning-based world models, observation-level generative world models, latent space world models, object-centric world models, robotics, autonomous driving, scientific discovery, virtual game simulation, GUI-based agents, and benchmark resources.

GitHub repository: [JiahuaDong/Awesome-World-Models](https://github.com/JiahuaDong/Awesome-World-Models)
